# Hire A Box — Design System & UI Guidelines

This document outlines the core brand tokens and typographic hierarchy used throughout the project to ensure a consistent, premium, and highly maintainable user interface.

## Brand Colors

*   **Orange (`#E8590C`)**: Used strictly for Call-to-Action (CTA) elements, primary buttons, and accents.
*   **Charcoal (`#2B2B28`)**: Used for primary text and dark UI sections. This is a warm charcoal, deliberately chosen instead of pure grey to harmonize with the overall warm palette.
*   **Kraft Brown (`#C8A47C`)**: Used for background surfaces and secondary UI elements.
*   **Deep Kraft (`#A6794E`)**: Used when more saturation is required for headers, borders, or dividers.
*   **Warm White (`#FAF6EF`)**: The primary page background color.

## Font Consistency & Typographic Hierarchy

The system enforces a strict typographic scale utilizing **Poppins** for structural elements and **Inter** for high-readability body copy.

*   **Hero headline (h1)**
    *   Font: Poppins 700
    *   Desktop: 52px | Mobile: 34px
    *   Line-height: 1.15
*   **Section heading (h2)**
    *   Font: Poppins 700
    *   Desktop: 34px | Mobile: 28px
    *   Line-height: 1.2
*   **Card / subsection title (h3)**
    *   Font: Poppins 600
    *   Desktop: 22px | Mobile: 20px
*   **Body copy**
    *   Font: Inter 400
    *   Size: 16–17px
    *   Line-height: 1.6
*   **Small / caption / label**
    *   Font: Inter 400
    *   Size: 14px
*   **Eyebrow / section kicker**
    *   Font: Poppins 600
    *   Size: 13px
    *   Styling: Uppercase, letter-spacing `0.08em`, colored in brand Orange (`#E8590C`).
*   **Buttons**
    *   Font: Poppins 600
    *   Size: 16px
*   **Pricing figures & stat numbers**
    *   Font: Poppins 700
    *   Styling: `font-variant-numeric: tabular-nums` (Ensures numerical columns align perfectly in pricing tables and carts).
